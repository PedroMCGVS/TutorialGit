# GIT FLOW 

## Os principais comandos:
 - git branch;
 - git checkout -b;
 - git merge;

## Como fazer a inicialização:
 - git init;

## O que é workFlow git:
 - É um fluxo de trabalho legado do Git que no começo era uma estratégia inovadora e revolucionária para gerenciar ramificações do Git.

## Como se integra ao repositório online:
 - git remote add origin <url>
